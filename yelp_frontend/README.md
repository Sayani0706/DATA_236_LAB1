# Yelp AI Prototype Frontend

This is the frontend for the **Yelp AI Prototype** built for **DATA 236 Lab 1**.  
It is a React-based web application that allows users to browse restaurants, view restaurant details, manage their profile and preferences, add restaurants, track activity, and interact with an AI assistant for restaurant recommendations.

## Features

- User authentication
  - Sign up
  - Login
  - Logout
- Home page with recommended restaurants
- Explore/search restaurants
- Restaurant details page
  - View restaurant information
  - View reviews
  - Write a review
  - Upload optional photo with review
- Profile page
  - Personal information
  - Country dropdown
  - State validation
  - AI dining preferences
- My Activity page
  - History tab
  - Favourites tab
- Add Restaurant page
- AI Assistant panel
  - Prompt-based restaurant recommendations
  - Quick suggestion buttons
  - Clear chat option

## Tech Stack

- React
- JavaScript
- Axios
- CSS / custom styling
- React Router
